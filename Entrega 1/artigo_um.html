Descrição:

Este projeto teve como escopo a aplicação teórica dos conceitos que estávamos aprendendo na disciplina de Grandes Desafios da Engenharia. Englobado pela tutoria, cada aluno escreveu textos de no máximo 7000 caracteres à respeito dos conceitos aprendidos.

Em contraposição ao senso comum de que o engenheiro não precisa saber escrever ou ainda que o engenheiro só precisa fazer contas, este projeto surgiu como continuação do primeiro para enrradicar definitivamente tais estigmas. O projeto anterior, o Artigo I, foi de certo modo uma entrega parcial deste último, considerando que era fortemente recomendado que corrigíssemos os erros identificados no primeiro com o intuito de entregar um projeto final de qualidade.
